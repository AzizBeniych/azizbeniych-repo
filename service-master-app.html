ServiceMaster Disaster Preparedness App (PWA)

I developed a fully responsive Progressive Web App for ServiceMaster to support users in securely storing emergency information such as contacts, insurance, roadside assistance, and critical documents. This mobile-first app was built to function seamlessly across web, iOS, and Android platforms and designed as a white-label solution for franchise deployment, including brands like Camping World and AutoNation.

Objective:  
To create a robust and scalable emergency preparedness application in time for Hurricane Preparedness Week 2025, giving users a way to organize and share vital disaster recovery information.

Project Overview:  
The application features a branded home screen, categorized dashboards, interactive forms, document uploads, contact management, and secure sharing functionalities. It includes an administrative backend for content control and licensing demos.

  Key Features Implemented:  
- Progressive Web App (PWA) built with Ionic + Angular.  
- Node.js + Express.js backend with MongoDB database.  
- Secure login via JWT-based authentication.  
- File uploads handled with AWS S3 integration.  
- Admin panel to manage categories, content, and media.  
- Clickable phone numbers and web links.  
- Contact slider with Add/Edit/Delete capabilities.  
- Share function to send specific category data to trusted contacts.  
- Multi-file uploads per category (up to 3 files).  
- SSL-secured deployment on AWS EC2 instance.  
- Mobile responsiveness tested across iOS, Android, and desktop.  
- Demo accounts provided for marketing, sales, and partner presentations.

Outcome:  
The app is actively being pitched to national brands and disaster-prep organizations as a white-label, license-ready solution. It enables fast and secure disaster readiness and recovery for homeowners, insurance partners, and service teams.
